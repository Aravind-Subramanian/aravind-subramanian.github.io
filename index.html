Event Driven Simulation

Implemented An Event Driven Cache Simulator System using different Cache Replacement Policies and Probabilistic Caching Methods.

Language and Tools : Python

  • In this Event Driven Simulation, the events are dynamically generated as the simulation proceeds
  • Each event has a time of occurence and the simulator processes the events in the order of their occurence times.
  • While, processing the events in the order of occurence, future events are generated and scheduled according to their times.
  • Implemented a priority queue to maintain all the outstanding events and access the next event efficiently.
  • Used different Probability distributions for each kind of events and implemented a cache system to keep track of certain events, instead of fetching directly from memory
  • Experimented with different kinds of caching algorithms and analysed the performance of simulation with different cache algorithms
  • Used numpy and seaborn libraries to visualize the analytics metrics.

Outcomes :Created and implemented an event-driven simulator for a network cache in Python and checked the performance and verified it for multiple test scenarios for different caching algorithms like LRU, MRU and LFU. Analyzed the behavior of the simulator on multiple probability distribution models with different parameters.
